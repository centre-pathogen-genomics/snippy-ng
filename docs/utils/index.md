# Utilities

Snippy-NG provides utility commands that support the main SNP-calling pipelines.

## Reference utilities

- [ref](ref.md): prepare a reference genome for Snippy-NG
- [gather](gather.md): discover sample files and write input for `snippy-ng multi`

## Alignment utilities

- [samclip](aln/samclip.md): filter clipped reads from SAM alignments using a
  boundary-aware terminal clipping rule
- [cnv](aln/cnv.md): estimate contig or feature copy numbers from aligned BAM
  or CRAM depth

## Report generation

- [tree](report/tree.md): render an interactive phylogenetic tree report
- [sample](report/sample.md): render an interactive per-sample variant report

See the individual command pages for usage examples and option details.
