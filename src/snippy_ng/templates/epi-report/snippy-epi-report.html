About

  • Multi-select samples: Click a row, map marker, or tree tip label to select across all views. Click additional items to add/remove them from the current selection set.
  • Clear selection quickly: Click the page background (outside the tree, map, and metadata panels) to clear all selected samples.
  • Explore the table: Sort columns, filter with header search, and page through rows.
  • Resize panels: Drag the row and column handles between panels to adjust layout.
  • Shortcuts: Press t (tree), m (map), d (data table), l (legend).
  • Colour by column legend: Use the palette button in the legend panel to choose a metadata column. The tree tips and map markers use the same colours. Click legend categories to select matching samples, and use Clear in the colour menu to reset colouring.
  • Add/merge metadata: Drag and drop CSV/TSV/JSON files onto the page (or use the load link). Rows are matched using one of id, sample_id, sample, or name. The chosen column values must correspond to tree tip labels (e.g. sample names) for linking to work.
  • Find changed metadata: Newly inserted rows and updated cells are highlighted in blue.
  • ISO subdivision mapping: If metadata includes an ISO 3166-2 column (e.g. iso3166-2), missing latitude/longitude values are auto-filled from built-in mapping.
